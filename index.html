Erin Mullaney @emullaney

Developer and Data Person.

I have a degree in computer science and have been developing applications and designing databases for fourteen years. I develop applications primarily using the following technology stack: Python, Django, jQuery, LESS, HTML5 and CSS3, with a focus on performance and elegant re-usable code.

I love what I do and want to use my design and programming skills to make life easier and improve the world.

Open-Source Projects I've worked on

RapidPro Community Portal

The Community Portal for RapidPro was built using the Wagtail CMS. I also wrote about adding pages outside of the CMS. Code can be found at https://github.com/rapidpro/rapidpro-community-portal.

TracPro: A Hosted Data Visualization Web App for RapidPro

I worked on TracPro, a dashboard add-on, custom designed for UNICEF. The add-on allows country offices to quickly build additional, more complex reports. I worked on updating all time-based charts and maps for the entire site utilizing HighCharts and HighMaps for visualizations.

MDN: Mozilla Developer Network

I've worked on the spam moderation pieces of the Mozilla Developer Network wiki in order to drastically cut down on the time it takes admins to clear out spam and band users. Work was forked off of the Mozilla Kuma repository.
